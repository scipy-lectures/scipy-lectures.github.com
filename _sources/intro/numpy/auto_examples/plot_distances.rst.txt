Distances exercise
==================

Plot distances in a grid





.. image:: /intro/numpy/auto_examples/images/sphx_glr_plot_distances_001.png
    :align: center





.. code-block:: python

    import numpy as np
    import matplotlib.pyplot as plt

    x, y = np.arange(5), np.arange(5)[:, np.newaxis]
    distance = np.sqrt(x ** 2 + y ** 2)
    plt.pcolor(distance)
    plt.colorbar()
    plt.show()
