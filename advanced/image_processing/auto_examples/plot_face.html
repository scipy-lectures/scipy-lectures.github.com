2.6.8.1. Displaying a Racoon FaceÂ¶

Small example to plot a racoon face.

from scipy import misc

f = misc.face()

misc.imsave('face.png', f) # uses the Image module (PIL)



import matplotlib.pyplot as plt

plt.imshow(f)

plt.show()
